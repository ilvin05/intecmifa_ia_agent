# INTEMCIFA AI AGENT 🤖

## Descripción

INTEMCIFA AI AGENT es un asistente inteligente desarrollado para el Instituto Tecnológico Michael Faraday (INTEMCIFA).

Este proyecto forma parte de mi proceso de aprendizaje sobre Agentes de IA, LangChain, LangGraph, RAG y Python.

---

## Objetivos

- Aprender a desarrollar Agentes de IA.
- Implementar RAG con documentos PDF.
- Construir una interfaz con Streamlit.
- Aplicar buenas prácticas de Git y GitHub.
- Documentar todo el proceso de desarrollo.

---

## Tecnologías

- Python
- Git
- GitHub
- Langchain
- Chroma
